# Peripheral Access Crate atsam3x8h for ATSAM3X8H 32-bit MCU from Atmel/Microchip

This crate contains autogenerated code to access peripherals of ATSAM3X8H generated
from official SVD files from Microchip website using
[svd2rust](https://github.com/rust-embedded/svd2rust/) tool.
